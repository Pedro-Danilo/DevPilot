# Plantillas operativas MIPSoftware

Esta carpeta contiene plantillas reutilizables para documentar productos, requerimientos, arquitectura, datos, APIs, UX, pruebas, seguridad, releases, operación, incidentes, mantenimiento y retiro.

Todas las plantillas deben usarse como artefactos vivos versionados en Git. Cuando un proyecto use IA, agentes, LLMs, RAG, memoria, tool calling o automatización inteligente, estas plantillas deben complementarse con los activos MIASI correspondientes.
