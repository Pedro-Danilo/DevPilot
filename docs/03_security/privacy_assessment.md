# Privacy Assessment

## Datos tratados en MVP

| Dato | Tipo | Sensibilidad | Política |
|---|---|---|---|
| Documentos de proyecto | Operativo | Media | Local-only |
| Reportes | Operativo | Media | No publicar sin revisión |
| API keys | Secreto | Alta | No usar en MVP |

## Regla

El MVP no debe procesar datos personales de terceros ni secretos reales.
