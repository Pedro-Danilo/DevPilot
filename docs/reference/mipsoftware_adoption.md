# Procedimiento de adopción de MIPSoftware

1. Abrir proyecto formal en `D:\Projects\DevPilot_Local`.
2. Crear artefactos mínimos pre-code.
3. Activar MIASI por tratarse de una plataforma agent-assisted.
4. Crear entorno Python local.
5. Ejecutar `pytest -q`.
6. Ejecutar `readiness-check`.
7. Solo después iniciar implementación incremental del CLI.
