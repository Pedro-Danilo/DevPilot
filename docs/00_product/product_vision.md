# Product Vision — DevPilot Local

## Problema

El desarrollo de software personal y freelance suele degradarse cuando los artefactos de ingeniería no quedan trazados: requerimientos incompletos, decisiones no documentadas, pruebas desconectadas, releases sin evidencia, seguridad aplicada tarde y documentación desactualizada.

## Visión

DevPilot Local será una plataforma local-first para asistir el ciclo de vida completo del desarrollo de software mediante validadores, plantillas, checklists, reportes y agentes controlados.

## Usuario primario

Ingeniero de sistemas/desarrollador independiente que quiere profesionalizar su flujo de trabajo y construir aplicaciones reales con criterios de producción.

## Propuesta de valor

DevPilot Local convierte MIPSoftware y MIASI en un flujo operativo: crea artefactos, valida completitud, detecta brechas, ejecuta quality gates y genera evidencia auditable.

## Resultado esperado del MVP

Un CLI local capaz de inicializar proyectos, validar artefactos mínimos, decidir si MIASI aplica y producir reportes de readiness.

## Métricas de éxito

| Métrica | Meta MVP |
|---|---:|
| Artefactos mínimos pre-code generados | 100% |
| Validación de checklist pre-code | PASS |
| Uso sin API keys | Sí |
| Ejecución local | Sí |
| Reporte JSON/Markdown | Sí |
