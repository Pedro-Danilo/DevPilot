# MVP Scope

## In scope

- CLI local.
- Inicialización de estructura documental de un proyecto.
- Validación de artefactos mínimos.
- Checklist pre-code.
- Detección de activación MIASI.
- Readiness report en JSON/Markdown.
- Tests herméticos con pytest.

## Out of scope

- SaaS multiusuario.
- UI web completa.
- Ejecución autónoma de cambios en repositorios.
- Integración obligatoria con APIs pagas.
- Despliegue cloud obligatorio.
- Agentes con acciones destructivas.

## Criterio de éxito MVP

El MVP está listo cuando pueda validar un proyecto local y producir un veredicto PASS/FAIL con evidencia trazable.
